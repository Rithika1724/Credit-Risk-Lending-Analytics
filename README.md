# 💳 Credit Risk & Lending Analytics Dashboard

An end-to-end banking analytics project focused on analyzing loan portfolios, borrower characteristics, lending performance, and credit risk using Python, SQL, Power BI, and Excel.

The project demonstrates a complete analytics workflow—from data preprocessing and exploratory analysis to SQL-based business querying and interactive dashboard development.

---

# 📖 Project Overview

Financial institutions rely on data-driven insights to evaluate borrower profiles, monitor lending performance, and manage portfolio risk.

This project analyzes historical banking loan data to answer key business questions related to:

- Loan disbursement trends
- Borrower demographics
- Credit grades
- Loan repayment performance
- Lending concentration
- Credit risk indicators

Python was used for data preprocessing and exploratory analysis, SQL for business querying, and Power BI for interactive dashboards.

---

# 🎯 Business Objectives

- Analyze loan portfolio performance.
- Understand borrower demographics and credit characteristics.
- Evaluate repayment trends and lending behavior.
- Identify high-performing borrower segments.
- Support data-driven lending decisions through interactive dashboards.

# 💡 Key Business Insights

- Lending activity is concentrated in a limited number of states.
- Moderate credit grades account for a significant share of the loan portfolio.
- Verified borrowers contribute a larger proportion of total repayments.
- Debt Consolidation remains the primary purpose of loan applications.
- Borrower income and employment history influence lending patterns.
- Credit grades exhibit varying repayment behavior and revolving balances.
